This repository is designed to help developers:

* Learn C++ from scratch
* Understand Object-Oriented Programming deeply
* Master STL and Generic Programming
* Build strong Data Structures & Algorithms skills
* Learn Modern C++ (C++11/14/17/20/23)
* Prepare for coding interviews
* Build real-world projects
* Understand memory management and performance optimization
* Write production-quality C++ code

#  Compilation Guide

## Linux / macOS

```bash
g++ main.cpp -o app

./app
```

### C++17

```bash
g++ -std=c++17 main.cpp -o app
```

### C++20

```bash
g++ -std=c++20 main.cpp -o app
```

---

## Windows

```bash
g++ main.cpp -o app.exe

app.exe
```
